# Project Nutrition
This was a project in the Technigo Bootcamp. The app scans barcodes on food packages to retrieve the product name and origion using React Redux and Thunks to fetch data from Open Food Facts.

## Tech

	• JavaScript ES6
	• React
	• React Redux
	• Redux Toolkit
	• Styled Components
	• JSX
	• Npm
